Files in alignment tool QUEUE

 all.sorted.bam the tophat alignment with multimappers
 
this file is split into:
uniq.sorted.bam contains only unique mappers.
Coverage profiles were generated from the uniq.sorted.bam file.
multi.sorted.bam contains only multi-mappers
 
 namesorted.bam is the uniq sorted bam file sorted by name.
This is important for a tool like htseq-count when using paired end reads.
 
 htseq-count There is already a file containing htseq-counts per gene called "htseq-count".
You could use these files to create an expression matrix for DE-tools. But htseq-count has many parameters, and maybe you want different ones.
 
 junctions_bed is a file containing the tophat discovered junctions.
 
 matched.sorted.bam contains the sequences that were filtered in the cleaning step against rRNA + ERCC the spike ins.
